# 四柱八字命理分析 Skill

## 概览
这是一份中国传统八字命理分析的完整操作指南，分为三个阶段：

## 第一阶段：信息收集（9步）
逐步收集用户的姓名、曾用名、阳历/农历生日、出生时辰、性别、出生地、在世状态，最后汇总确认。关键原则是"一步一步来"，避免一次性提出所有问题。

## 第二阶段：排盘计算
基于经典规则计算：
- **年柱**：以立春为分界线（通常阳历2月3-5日）
- **月柱**：以节气为分界线，使用"年上起月法"
- **日柱**：查万年历或推算法
- **时柱**：根据时辰和"日上起时法（五鼠遁）"确定

生成四柱表、十神分析、大运排列等。

## 第三阶段：综合分析
涵盖日主旺衰、十神含义、五行平衡、格局判定、大运/流年分析，以及历史事件校准（让用户验证过去发生的事件是否符合推算）。

## 核心约束
- 引用《穷通宝典》《三命通会》《滴天髓》等9部经典著作
- 处理边界情况（时辰未知、闰月出生、夜子时等）
- 保持中性语气，强调分析仅供参考
- 参考文件：`wuxing-tables.md`、`shichen-table.md`、`dayun-rules.md`、`classical-texts.md`

此skill在用户提及"八字""算命""四柱"等关键词时自动触发。
